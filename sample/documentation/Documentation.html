Markup Markup customization and highlights

This theme is a fixed layout with one column. All of the information within the main content area is nested within a div with an class of "container".

General Markup

The general template structure is the same throughout the template. Here is the general structure.

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01//EN"
"http://www.w3.org/TR/html4/strict.dtd">

<html lang="en">

    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
        <title>Jodstrap</title>
        <meta name="author" content="Saltflakes">
        <!-- CSS Styles -->
    </head>

    <body>
        <!-- Navigation -->
         
        <div class="container">
        <!-- content area -->
        </div>

        <!-- Footer -->

        <!-- Javascript Files -->

    </body>

</html>

Nagigation Top Border Color

If you would like to edit the color of the animated navigation border you would do the following:

.backLava {
border-top: 4px solid #FF8900;
}

Customize Slider

If you want to change the slider's speed transition, open up the file standard.js and change the following code: 

   // camera slider options:
    if ($('#camera_wrap_1').length > 0) {
        $('#camera_wrap_1').camera({
            thumbnails : true,
            height : '35%',
            loader : 'none',
            opacityOnGrid : false,
            onLoaded : function() {             
            }
        });
    }
